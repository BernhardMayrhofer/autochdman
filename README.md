# autochdman
chdman - MAME Compressed Hunks of Data (CHD) manager 0.242 (unknown)

## Install

```bash
#install chdman
apt install mame-tools perl
```

## Run

### File Structure
```
your_7z_files_directtory\
    file1.7z
    file2.7z
    ...
    autochdman.pl
```

### Start Conversion
```bash
your_7z_files_directtory$ perl autochdman.pl
```
